Turf.js (distanceToRadians)

This command converts any measurement of distance to radian angles. It takes a numeric input argument and a string indicating the input units (degrees, radians, miles, kilometers, inches, or yards).

Turf.js (squareGrid)

This command creates a squre grid from a bounding box. The input arguments are the extents of the box, bbox (max and min lat and lon), cellSize (width of each cell), units (cellSize units in degrees, radians, miles, or kilometers), and completelyWithin (boolean argument either True or False which adjusts width and height cellSize to fit in the bounding box).

Turf.js (bearing)

This command takes two points and caluculates the bearing between them. The input arguments are staring and ending points with a boolean argument whether the final brearing is true of false. This command can be useful for navigation purposes (airplanes and ships).

Turf.js (random)

This command is helpful for testing purposes. It creates random GeoJSON data (both points and polygons). The input arguments are a string either 'point' or 'polygon', count (number of data to be generated), and options (relates to the features).

Proj4js (proj4.def)

Allows user to define and georeference a projection as a string. This command can be called using the name and projection and using an array of projections.
